## Application Overview (Erupass)

Erupass:
- **Offline-first** çalışan
- **Local encrypted vault** kullanan
- **Master password** ile korunan
- Electron + React + Python backend mimarisine sahip
bir masaüstü parola yöneticisidir.

Bu repository yalnızca **tanıtım ve indirme** amaçlıdır.
